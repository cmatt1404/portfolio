PromotionalPoster

Illustration/Composition

This project was created for a class where we had a unit on creating branding for a local business. As the image and title show, I chose a local pizza shop.

This piece was created in December 2022. This was the project for my class and we were tasked with craeting a promotinal graphic for an event of our choosing.

This work was made using a mix of both Illustrator and Photoshop. I used Illustrator to create the two characters at the botom in a style that I have come across and I liked. Once I had those assets I built everything else in Photoshop where I was able to lay everything out and tweak the hierarchy of the text as well as the illustrations.
