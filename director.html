Oscars 2023

Photo
Manipulation

This project was an assignment in one of my classes where we focused of various selection tools in Adobe Photoshop.

This piece was created in February of 2023. As mentioned above the main objective of this project was using different selection tools and focusing on creating a cohesive composition.

This project was assigned around the time that the Oscar nominees were being announced and I was tasked with creating a magazine centerfold for the nominees. The left half of the composition was created by our professor and I was supposed to create something that was supposed to thematically match.
